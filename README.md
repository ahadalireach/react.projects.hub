# React Projects Hub

**React Projects Hub** is a web application showcasing 10 mini React projects, featuring interactive and UI-focused components.

## Technologies Used

- ReactJS, React Router DOM, Framer Motion, TailwindCSS

## Features and Projects

The application includes the following projects:

| #   | Project Name          | Category    | Description                                                                                 | Icon |
| --- | --------------------- | ----------- | ------------------------------------------------------------------------------------------- | ---- |
| 1   | Accordion             | UI          | A simple FAQ section where you can click questions to see the answers.                      | 🔽   |
| 2   | Grocery Manager       | Interactive | A simple and interactive grocery list manager where you can add, edit, and remove items.    | 🛒   |
| 3   | Header Navigation     | UI          | A responsive navigation bar that includes a logo, menu links, and social media icons.       | 📑   |
| 4   | Random Text Generator | Tools       | A tool that generates customizable random paragraphs to make your content more interesting. | 🔤   |
| 5   | Restaurant Menu       | Interactive | A platform developed to showcase a restaurant's menu with dynamic categories and filtering. | 🍽️   |
| 6   | Birthday Tracker      | UI          | A simple birthday tracker that shows a list of birthdays and allows clearing/restoring it.  | 🎂   |
| 7   | Stripe Theme Design   | Interactive | A sleek, modern interface inspired by Stripe’s UI, featuring smooth hover navigation.       | 💳   |
| 8   | Sidebar with Modal    | Interactive | A sidebar that can be toggled with a modal for easy navigation, offering an interactive UI. | 🗂️   |
| 9   | Reviews Slider        | UI          | A dynamic customer reviews slider with smooth transitions and star ratings.                 | ⭐   |
| 10  | Shopping Cart         | Interactive | A fully functional shopping cart where users can add, remove, and update items.             | 🛍️   |

## Getting Started

### Prerequisites

- Node.js and npm installed
- VS Code or any other code editor

### Setup Instructions

1. **Clone the Repository**

   ```bash
   git clone https://github.com/ahadalireach/react.projects.hub.git
   ```

   Unzip the downloaded file if needed.

2. **Open with VS Code**

   Open the project directory in VS Code or your preferred code editor.

3. **Install Dependencies**

   Navigate to the project directory and install the dependencies:

   ```bash
   cd react.projects.hub
   npm install
   ```

4. **Run the Development Server**

   Start the development server:

   ```bash
   npm run dev
   ```

5. **Access the Application**

   Open your browser and navigate to [http://localhost:5174](http://localhost:5174) to view the application.
